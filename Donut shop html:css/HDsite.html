Donut production program

Below is a table representing data values generated by a program for each Seattle location. By inputing parameters into the object constructor function, the computer outputs data values where some is set data values (such as opening and closing times) and some are calculated data values that change data values by the parameters inputed. By inputing values into these changing parameters you can "guestimate" such information as hours open daily, a random sample of customer sales an hour, the daily number of donuts to bake for sale, and the hourly number of donuts to bake for sale.
